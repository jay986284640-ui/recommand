你是一个生活服务领域的对话生成专家。你的任务是：给定了目标参数（params），生成一段自然的用户多轮对话，以及助手的引导语。

# 目标参数（从这段对话中应能提取出以下参数）
{target_params}

# 字段定义（供参考，帮助你写出符合参数约束的对话）
{field_definitions}

# 要求
1. 生成 {target_turns} 轮对话，用户必须说第一句
2. 用户的表述要**自然口语化**，每次生成要使用不同的句式、语气、用词，避免重复
3. guide_text 是助手对用户需求的**总结确认**，用自然语言复述用户的约束条件
   - 格式多样化：有时简洁确认，有时详细总结，有时追问缺失维度
   - **不要**在 guide_text 中具体推荐某家店（如"推荐XX店"）
   - **不要**每句都用"好的/明白了"开头
   - 示例多样风格："帮您确认一下，您要的是...对吧？"、"收到，我记一下：品类咖啡、人均50以内"、"总结一下您的需求..."
4. 严格 JSON 输出

# Output Format
{{
  "messages": [
    {{"role": "user", "content": "用户的第一句话"}},
    {{"role": "assistant", "content": "助手的追问或确认"}},
    {{"role": "user", "content": "用户补充或确认需求"}},
    {{"role": "assistant", "content": "..."}}
  ],
  "guide_text": "助手对用户需求的总结/确认（不推荐具体店铺）"
}}
